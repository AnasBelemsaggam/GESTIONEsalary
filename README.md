# Personal Finance Tracker

Desktop application built with Java Swing and SQL Server.

## Features

- Login / Register
- Admin and User roles
- Manage users
- Add salary
- Salary history
- Add expenses
- Expense history
- Update / Delete expenses
- Monthly report
- Dashboard with real database statistics
- Spending by category pie chart

## Technologies

- Java
- Java Swing
- JDBC
- SQL Server
- DAO Architecture
- UML
